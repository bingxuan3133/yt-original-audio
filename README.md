# YouTube Original Audio Switcher

Automatically switches YouTube videos (including Shorts) to the **original audio track** when they load.

## Features

- Auto-switches to the original audio track on YouTube videos.
- Supports both standard videos and Shorts.
- Works with YouTube’s SPA navigation (clicking links without full page reload).
- Lightweight and simple — no extra permissions required.

## Installation

1. Clone or download this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (top-right toggle).
4. Click **Load unpacked** and select this extension's folder.

## Usage

- Once installed, the extension runs automatically when you open a YouTube video.
- If an original audio track is available, it will switch to it automatically.
- Works for standard videos and Shorts.
